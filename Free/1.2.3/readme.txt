### Facebook Popup Likebox

* [Facebook Popup Likebox Pro](http://ays-pro.com/index.php/wordpress/facebook-popup-likebox)
* [Facebook Popup Likebox Demo](http://demo.ays-pro.com/wp/2016/06/17/ays-facebook-popup-likebox/)

https://www.youtube.com/watch?v=CJa2La6BlbU

With the help of this  amazing plugin you can promote your Facebook page and add number of Likes , which is very important today. When someone visits your website, plugin opens popup box with your facebook page likebox. It has many types of parameters, which make it multifunctional and useful. You can choose the page of your website in which  popup box will be shown - 

* Home Page
* All Pages 
* All Posts 
* Archives


Also it's possible to choose how frequent likebox  will be appeared for the same website visitor. User can configure dimensions of likebox with choosing width and height from backend. It has auto close ability duration of which must be chosen by you. Also we give you possibility to give interesting subject for likebox, which will make your likebox more attractive.

== Installation ==

Thank you for your interest in Ays Facebook Popup Likebox.
There are two ways to install Ays Facebook Popup Likebox: the easy way, when you install the Likebox from your WordPress dashboard, and the second not so easy way, when you install it from WordPress.org. 
From your WordPress Dashboard
1) Visit Plugins > Add New.
2) Search for Ays Facebook Popup Likebox.
3) Activate Ays Facebook Popup Likebox from your Plugins page.
4) Visit Settings > Ays Facebook Popup Likebox.

From WordPress.org
1) Download Ays Facebook Popup Likebox.
2) Upload the ays-facebook-popup-likebox.zip content directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...).
3) Activate Ays Facebook Popup Likebox from your Plugins page.
Once activated configure any options as desired and you can enjoy our Ays Facebook Popup Likebox.



== Wordpress Facebook Popup Likebox step by step guide ==

Installation
There are two ways to install Ays Facebook Popup Likebox: the easy way, when you install the Likebox from your WordPress dashboard, and the not so easy way, when you install it from WordPress.org. 

* 1.1 The easiest way to enjoy Facebook Popup Likebox:
* 1.1.1	Login to your WordPress dashboard
* 1.1.2	Go to Plugins 
* 1.1.3	Add New
* 1.1.4	Search for Ays Facebook Popup Likebox
* 1.1.5	Click to install

* 1.2 The second way:
* 1.2.1Download the zip file from https://wordpress.org/plugins/ays-facebook-popup-likebox/
* 1.2.2 Go to Plugins 
* 1.2.3 Add New 
* 1.2.4 Upload plugin
* 1.2.5 Choose file ays-facebook-popup-likebox.zip
* 1.2.6	Click to install

* 1.3 In order to install the AYS Facebook Popup Likebox from Wordpress.org you must unzip the archive and copy the ays-facebook-popup-likebox folder into your plugins folder (\wp-content\plugins). 
After one of these steps you can activate the plugin from the Plugins menu.
Once activated configure any options as desired and you can enjoy our Ays Facebook Popup Likebox.

* 2.2 Styles
* 2.2.1 Border color
With this option you choose the color of the border around the likebox. You are free to set any color you like.
* 2.2.2 Background color
This option allows you to set the background color. This is the color in which the likebox appears. This option gives you a wide range of colors to choose from.
* 2.2.3 Border radius
This is the option which lets you adjust the level of roundness of the likebox borders. 
* 2.2.4 Border thickness
You are allowed to choose not only the color of the border, but also its thickness. This option is here to help you out. You set the thickness with pixels.  
* 2.2.5 Opacity background color
After giving the Likeboxyour preferred design you can move to the opacity background. Here you can change the background color. You can choose any color you like.
* 2.2.6 Opacity level
With this option you set the transparency level of the background. It is a scale of 0-1. If you want it entirely opaque, you just put a 1 in there. But if you want it to be very see through then you put 0.1.
* 2.2.7Close button color
And last but not least you can set the color of the close button. The color is again chosen from a wide range.
* 2.2.8 Custom Css
If you need to configure the CSS style of Facebook Popup Likebox on your own, then you can just add any style you wish toCustom Css fieldand make your modifications there.
